Creating your first docker container

Pre-requisites

  • Docker Installation
  1. Initiate your project. Create a folder called my-first-docker-project and contents folder:
mkdir my-first-docker-project contents
  1. Create a file called Dockerfile:
touch Dockerfile
vim Dockerfile
  1. Add the following contents:
FROM nginx:latest

COPY contents /usr/share/nginx/html
  1. Create index.html file inside the contents folder for your static page that will be served by NGINX proxy. Run the command via:
vim contents/index.html

Then, add the following content:

<!DOCTYPE HTML>
<head>
    <title>My First Container</title>
</head>
<body>
    <center>
    <h1>Welcome to my First Container</title>
    </center>
</body>
</html>
  1. Build your container using the following commands. For example: docker build -t <your-target-image-name> .
docker build -t my-first-container:latest .
  1. Run your container:
docker run -it --rm -p 8080:80 my-first-container:latest
